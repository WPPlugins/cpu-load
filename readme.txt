Ein Dashboard-Widget, welches Daten über die CPU-Auslastung des Servers sowohl in Text als auch in grafischer Darstellung zeigt.

== Description ==

[Support](mailto:info@patrick-designs.de)  


CPU-Load ist ein Dashboard-Widget, welches die aktuelle CPU-Auslastung des Servers anzeigt.  
Des-weiteren werden Tagesdurchschnittswerte, sowie Gesamtdurchschnittswerte und Höchstwerte angezeigt.  
Unterstützt wird das ganze durch einen Graphen.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin  
manually.  
  
1. Upload \`cpu_load.php\` to the \`/wp-content/plugins/\` directory  
2. Activate the plugin through the 'Plugins' menu in WordPress  
3. Now you will see the Widget on your Dashboard  
  
Sie koennen das Plugin mit dem integriertem Wordpress-Installer installieren, oder es manuell installieren  
1. Lade \`cpu_load.php\` in das Verzeichnis \`/wp-content/plugins/\` hoch  
2. Aktiviere das Plugin  
3. Nun sehen sie das Plugin auf ihrem Dashboard
== Frequently Asked Questions ==

F: CPU Load zeigt einen Wert ueber 100% an, wie kann das moeglich sein?

A: Falls ein Wert ueber 100 % angezeigt wird, bedeutet dies, dass der Server nicht in der Lage war, die Anfragen ohne Zeitverzoegerung zu verarbeiten
